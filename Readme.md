# Reed Catalog
A program for managing bassoon reeds
## Database structure
Tables: 
### Log
- date-time
- event
- order no.
- transaction
- actual wealth
### Reed
- id
- batch no.
- condition
- sell price
- order no.
### Batch
- id
- batch name
- date of buying
- maker
- quantity
- total buy price
- description
### Order
- id
- date of purchase
- customer
- total price
- amount to pay
